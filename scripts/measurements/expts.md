Experimental data
=================

Report data in either polar or cartesian coordinates, whichever is convenient but make sure to report the units.

- cal.: calibrate
- off.: offset
- st.: start
- pos.: position

- P: Pole
- W: Wall

- V/v: Violet Turtlebot
- I/i: Indigo Base Station
- B/b: Blue Turtlebot
- G/g: Green Turtlebot
- Y/y: Yellow Turtlebot
- O/o: Orange Turtlebot
- R/R: Red Turtlebot

- _ft_: feet
- _in_: inches
- _m_: metres
- _cm_: centimeters
- _rad_: radians
- _deg_: degrees
